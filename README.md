# Blazor Custom Component

This is the [first part](https://www.puresourcecode.com/dotnet/blazor/write-a-reusable-blazor-component/) of a Blazor article series to tell how to create a reusable Blazor component. So, we can use them in our Blazor application independently of the hosting model ([Web Assembly](https://www.puresourcecode.com/tag/blazor-webassembly/) or [Server-side](https://www.puresourcecode.com/tag/blazor-server)).
